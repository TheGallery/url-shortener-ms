URL Shortener Microservice

Endpoints:

https://pure-reef-73475.herokuapp.com/[urlId]
where [urlId] is the short url id. If this is valid, it will redirect to the original url.

https://pure-reef-73475.herokuapp.com/generate/[url]
where [url] a valid url to be shortened.

Example usage:

https://pure-reef-73475.herokuapp.com/generate/http://www.twitter.com
https://pure-reef-73475.herokuapp.com/wod3r

Example output:

{"original_url":"http://www.twitter.com","short_url":"https://pure-reef-73475.herokuapp.com/wod3r"}
